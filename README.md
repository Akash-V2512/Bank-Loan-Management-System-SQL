# Bank-Loan-Management-System-SQL
About the Project
This project is a SQL-based Bank Loan Management System designed to manage customer details, loan applications, approvals, and credit validation.
The system uses SQL queries, joins, stored procedures, and triggers to simulate real-world banking operations and ensure proper loan processing rules.

Project Objectives

Manage customer and loan data
Validate loan approvals based on credit score
Track loan status and details
Perform analytical queries on loan data

Technologies Used

MySQL
SQL Queries
Joins
Subqueries
Stored Procedures
Triggers
Views

Key Features
Customer and Loan tables with relationships
Trigger for credit score validation
Stored procedures for loan approval process
Aggregate queries for reporting
Join queries for data analysis

Key Learnings
Implemented real-world banking logic using SQL
Understood how triggers enforce business rules
Practiced joins, subqueries, and stored procedures

Strengthened database design concepts
